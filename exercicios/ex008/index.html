Principais Formatações

Negrito/ Destaque

Estou aqui testando como usar Negrito para destacar as frases

Italico/ Enfase

Neste paragrafo vou estar usando o termo EM que da origem a italico

Texto destacado

podemos observar que usaremos o termo MARK para deixar a frase marcada

Texto excluido

Texto que sera excluido sendo assim usamos o termo DEL para que leiam mas não considere

Texto inserido

Tambem podemos utilizar o termo INS para que seja inserido e considerado uma frase

Texto sobrescrito

Podemos utlizar o termo SUB para adicionar um texto em cima exemplo: x30+10

Texto subscrito

Também podemos adicionar o termo SUB que usamos para impor uma palavra abaixo da frase exemplo: H2O
